# Interactive Reptile Cursor 🐍

An interactive reptile cursor animation built using JavaScript and HTML Canvas.  
The creature dynamically follows the mouse using segment-based motion and inverse kinematics principles.

## 🚀 Features

- Real-time mouse tracking
- Smooth segment-based animation
- Canvas rendering using JavaScript
- Dynamic creature movement
- Customizable size, legs, and structure

## 🛠️ Technologies Used

- JavaScript
- HTML5 Canvas
- CSS
- VS Code Live Server

## 📂 Project Structure

project-folder
│── index.html
│── script.js
│── README.md


## ▶️ How to Run

1. Clone the repository

git clone https://github.com/your-username/interactive-reptile-cursor.git


2. Open the folder in VS Code

3. Right click `index.html`

4. Click **Open with Live Server**
